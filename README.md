# Pet Game Project

A simple Godot 4.5.1 project featuring coins, gems, and a food system for feeding your pet.

## üöÄ How to Run the Game

### 1. Install Godot
- Download the latest **Godot 4.5.1 (Mono or Standard)** from [https://godotengine.org](https://godotengine.org).
- Install and launch Godot.

### 2. Open the Project
- In the Godot Project Manager, click **Import**.
- Navigate to the folder containing this project.
- Select the `project.godot` file.
- Click **Import & Edit** to open the project in the editor.

### 3. Play the Game
- Once the editor loads, press **F5** or click the ‚ñ∂Ô∏è **Play** button in the top-right corner.
- The game will run in a new window.

## üéÆ Gameplay Instructions

- **Top Bar**  
  Displays your current **coins** and **gems**.  
  Use the settings menu to open **Settings**, **Help**, or exit the game.

- **Pet Area**  
  Your pet is displayed here.

- **Feed Control**  
  Below the pet, click the **Feed** button to open the food popup.  
  Select a food item (e.g., Apple, Meat, Fish, Pizza) to feed your pet.  
  Food counts decrease when consumed and are saved automatically.

- **Saving & Loading**  
  Coins, gems, and food inventory are saved to `user://save_data.json`.  
  When you restart the game, your progress is restored.
